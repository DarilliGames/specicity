Specificity, what is it?

1 - A CSS Rule assigned to a bare element Gets 1 Point.

2 - Every Additional Element adds 1 Point. This is DIV P. Adding 2 points in total. Note: Styling features that are not over ridden are kept! The Font-Size remains the same as we are only overriding the color

10 - Classes add 10 points to specificity. So while this is also a p element, the class will over-ride the p.

10 - If 2 rules have the same points, the latter in CSS will apply.

20 - Calling Elements within Elements .open .red

20 - Adding Rules to elements .open.red

20 - We want this to be brown. But our TryBrown CSS Rule is only has a strength of 13 Points (body div p.trybrown)
While the open red rule is worth 20!

23 - This is worth 23 points(body div p.red.trybrown)

100 - ID's Cut through a lot of specificity! This beats the 23 points with a staggering 100! WOW!

1000 - But inline styling beats absolutely everything!
